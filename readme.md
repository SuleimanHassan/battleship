# Battleship: Cloud Computing project
## By Ezzat Chebaro and Sulieman Hassan
Note that services need to be hosted at first (azure fuctions)
## Starting the game:
1. Open login.html in your browser and click create game.
2. Player 2 should now open login.html and join the game by entering the gameID visible on the bottom left corner of Player 1's screen and clicking on Join.
3. Both players should set up their board using the form on the top right corner.
4. Players now take turn selecting a cell on the enemies board and clicking Shoot.
